# Task Management Full Stack App

A full stack Task Management application built with **Node.js, Express, PostgreSQL, JWT Authentication, and React (Vite)**.

This project demonstrates secure authentication, protected APIs, and CRUD operations for managing tasks.

---

## Tech Stack

### Backend
- Node.js
- Express.js
- PostgreSQL
- JWT Authentication

### Frontend
- React
- Vite
- Axios

---

## Features

- User Registration
- User Login with JWT
- Create Task
- Update Task
- Delete Task
- Protected Routes
- Dashboard showing user tasks
